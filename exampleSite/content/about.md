Plural Newspaper is a Hugo theme for blogs and informational websites.

It is designed to be accessible, fast, and lightweight, with a clean newspaper-style layout.

## Goals

- Accessible by default (WCAG 2.1 AA baseline)
- Functional without JavaScript
- Responsive for mobile and desktop
- Configurable color schemes: light, dark, and high contrast
