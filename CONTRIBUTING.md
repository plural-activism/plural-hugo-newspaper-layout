# Contributing

Thank you for contributing to this theme.

## Workflow

1. Fork the repository and create a topic branch.
2. Keep changes focused and small.
3. Open a pull request with a clear summary of the change.

## License expectations

By contributing to this repository, you agree that your contributions are licensed under the GNU Affero General Public License v3.0 or later (AGPL-3.0-or-later).

## License headers

Use SPDX headers in source files that support comments:

- `SPDX-License-Identifier: AGPL-3.0-or-later`

Examples:

- HTML templates / JavaScript / CSS: `<!-- SPDX-License-Identifier: AGPL-3.0-or-later -->` or `/* SPDX-License-Identifier: AGPL-3.0-or-later */`
- Shell scripts / YAML / TOML (when adding comments): `# SPDX-License-Identifier: AGPL-3.0-or-later`

Do not add license headers to generated files or vendored third-party code.
